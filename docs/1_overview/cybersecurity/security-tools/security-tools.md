# Security Tools

The technical arsenal of digital defense.

Securing a modern environment requires a suite of specialized tools, from network defenses like firewalls to advanced monitoring systems like SIEM. This section covers the technical tools and operational processes used to keep data safe.

## The Catalog of Security Tools

### 1. [[siem|SIEM (Splunk & Chronicle)]]
The centralized monitoring systems that collect and analyze security logs in real-time.

### 2. [[playbooks|Security Playbooks]]
Standardized manuals for handling incidents and preserving digital evidence.

### 3. [[firewalls|Firewalls]]
The "building inspectors" of network traffic, controlling what enters and leaves a network.

### 4. [[vpn|VPN (Virtual Private Network)]]
Tools for creating secure, encrypted tunnels for remote access and privacy.

### 5. [[antivirus|Antivirus & Anti-Malware]]
Software designed to detect and eliminate malicious code pattern on endpoints.

### 6. [[ssh|SSH (Secure Shell)]]
The primary tool for managing remote servers securely over an encrypted connection.

### 7. [[beyondtrust|BeyondTrust (PAM)]]
Privileged Access Management tools for controlling high-level administrative credentials.
