# Demarcation Problem

"The Line in the Sand"—the philosophical question of how to tell the difference between **Science** (Astronomy, Biology) and **Non-Science** or **Pseudoscience** (Astrology, Alchemy).

The **Demarcation Problem** is the struggle to define the "Rules of the Club" for science. It sounds simple, but there is no single rule that perfectly separates the two. If you say science is about "Observation," well... an astrologer observes the stars every night. If you say it's about "Complexity," alchemy is incredibly complex. 

Think of it like **Detecting a Fake ID**:
*   **The Bouncer (The Philosopher):** Is standing at the door of the "Hall of Truth." 
*   **The Entrant (The Theory):** Presents their "ID" (their research and data).
*   **The Problem:** Some fakes are so good they look like the real thing. Some real IDs look messy. The Bouncer needs a set of unmistakable "Security Features" to tell which one is which. 
*   **The Stakes:** If we let in the "Fakes" (Pseudoscience), we waste money on magic potions instead of medicine. If we kick out the "Truth" by mistake, we might miss the next big discovery.

## The 3 Key "Security Features"
1.  **[[falsifiability|Falsifiability]] (Popper's Rule):** A real ID has a "Kill Switch." If there is no possible way to prove the theory wrong, it isn't science—it’s just a belief.
2.  **[[scientific-method|The Method]]:** Does it follow the loop? (Hypothesis $\rightarrow$ Test $\rightarrow$ Update).
3.  **[[scientific-consensus|The Crowd]]:** Does it stand up to the scrutiny of other experts? If only one "Guru" says it’s true, be careful.

## FAQs

*1. Is "Non-Science" bad?*
**Absolutely not.** Art, Music, Religion, and Philosophy are all "Non-Science," but they are essential parts of life. The danger is **Pseudoscience**—which is "Non-Science" that **pretends** to be science to trick you into buying something or believing a lie.

*2. Why do we need to solve this?*
Because it decides what gets taught in schools, what gets government funding, and what evidence is allowed in a courtroom. We need to know where the line is to protect society from nonsense.
